Shoes Hub Premium Website

How to run:
1. Open terminal in this project folder.
2. Run: npm install
3. Run: npm install react-router-dom react-icons
4. Run: npm run dev

Pages:
- Home
- Womens
- Mens
- Kids
- Sales

Included features:
- React Router page navigation
- Blur / hide-on-scroll navbar
- Mobile menu
- Product filters on the left side
- 12 product cards on category pages
- Add to cart with quantity and total calculation
- Wishlist count
- Checkout drawer and payment modal
- Toast alert / success messages
- Static WhatsApp support button
- Scroll reveal animations
- Delivery details and payment method options

Banner image:
- Replace src/assets/Poster1.png or update the image import in src/Packages/Home/Home.jsx
