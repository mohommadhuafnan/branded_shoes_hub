This is the cleaned source project for Shoes Hub.

Changes made:
- Switched BrowserRouter to HashRouter for GitHub Pages compatibility.
- Set Vite base to ./ so assets load correctly.
- Removed node_modules and old dist from the ZIP.

How to run locally:
1. npm install
2. npm run dev

How to build:
1. npm run build
2. Upload the contents of dist to GitHub Pages root.
